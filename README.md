# Linux Diagnostic for Beginners

## Description
Linux Diagnostic for Beginners is an interactive CLI tool designed to help new Linux users
understand the current state of their system.

It provides clear diagnostics and explanations for common system components such as CPU,
memory, disk, network, users, and running services.

This project focuses on simplicity, clarity, and learning.

## Target audience
- Linux beginners
- Students learning system administration
- Junior system administrators
- Anyone wanting a quick Linux health check

## Features
- Interactive CLI menu
- System information overview
- CPU, memory, and disk diagnostics
- Network connectivity check
- User and permission overview
- Running services check
- Beginner-friendly explanations

## Use cases
- First Linux system check
- Learning basic Linux commands
- Troubleshooting common issues
- Teaching Linux fundamentals
